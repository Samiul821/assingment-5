Question-1: What are the different ways to select an element in the DOM?

DOM এ একটি উপাদান নির্বাচন করার বিভিন্ন উপায় হলো: getElementById দিয়ে, getElementByClassName দিয়ে, getElementByTagName দিয়ে, querySelector দিয়ে এবং querySelectorAll দিয়ে।

Question-2: What is the difference between innerHTML, InnerText and textContent?

innerHTML, InnerText এবং textContant মধ্যে পার্থক্য হলো : innerHTML শুধু html কোড রিটার্ন করে, innerText শুধু visible টেক্সট রিটার্ন করে এবং textContant শুধু টেক্সট কন্টেন্ট রিটার্ন করে যদি সেটা visible না থাকে।

Question-3: What is event delegation in the DOM?

event delegation হল একটি JavaScript টেকনিক, যেখানে আমরা একটি parent এলিমেন্টে ইভেন্ট লিসেনার অ্যাটাচ করি, যাতে তার child এলিমেন্টগুলোর ইভেন্টও পরিচালিত হয়।

Question-4: What is event bubbling in the Dom?

Event Bubbling হল একটি ইভেন্ট প্রথমে নির্দিষ্ট এলিমেন্টে ঘটে এবং তারপর তার parent এলিমেন্টের দিকে উপরে উঠতে থাকে যতক্ষণ না document পর্যন্ত পৌঁছে।
